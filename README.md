## ⚽ FIFA World Cup 2026 Dashboard

A clean, responsive, and real-time dashboard built to track the 2026 FIFA World Cup matches. This project was developed out of a passion for football to provide fans with an easy way to stay updated with the tournament schedule, scores, and standings.

### Key Features

- **Match Schedule:** Track all match dates and times with a sleek, user-friendly interface.
- **Scorer Details:** View goal scorers for both home and away teams directly on the match card.
- **Group Standings:** Access comprehensive, up-to-date group stage standings tables.
- **Smart Search:** Easily find your favorite team's upcoming or finished matches.
- **Timezone Support:** Adjust match timings according to different timezones for a seamless experience.
- **Dark/Light Mode:** Toggle between themes for optimal viewing comfort.

### Tech Stack

- **Framework:** [Next.js](https://nextjs.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Deployment:** [Vercel](https://vercel.com/)

Live: https://fifa-wc-2026-delta.vercel.app/
